# 课程说明

## 课程目标

本课程训练学生从“能写代码”提升到“能控制复杂软件系统”。课程关注需求、过程、项目管理、建模、架构决策、质量保障和交付运维。

| 课次 | 主题 | 对应教材 |
|---|---|---|
| 第01讲 | 从能写代码到能控复杂系统 | 第1章 从能写代码到能控复杂系统 |
| 第02讲 | 软件过程模型与开发节奏选择 | 第2章 软件过程模型与开发节奏选择 |
| 第03讲 | 项目管理、可行性分析与风险控制 | 第3章 项目管理、可行性分析与风险控制 |
| 第04讲 | 需求工程：从用户声音到可验证规格 | 第4章 需求工程：从用户声音到可验证规格 |
| 第05讲 | 架构表达与系统建模 | 第5章 架构表达与系统建模 |

## 学习方式

- 课前阅读对应教材章节；
- 课堂结合 PPT、案例和任务理解方法；
- 课后完成作业并推进课程项目；
- 期末复习以教材第1-8章、PPT 和项目材料为主。
