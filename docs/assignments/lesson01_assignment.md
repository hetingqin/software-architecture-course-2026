# 第01讲课后作业

## 作业标题

从能写代码到能控复杂系统

## 作业目标

完成课程项目分组与选题，为后续 M1 打基础。

## 提交内容

<div class="assignment-box">
完成课程项目分组（4-6人），确定一个可做的软件系统题目，提交选题说明和角色分工。
</div>

## 提交格式

- 建议提交 Markdown、PDF 或课程要求指定格式；
- 小组作业需写清成员分工；
- 图表、模型和 ADR 等材料应有标题、说明和必要解释。

## 截止要求

以任课教师课堂通知或课程平台通知为准。
