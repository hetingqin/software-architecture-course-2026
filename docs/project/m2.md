## 评价关注点

这里列出的是学生可见的质量要求，不是教师阅卷标准。

- 是否表达清楚问题、约束和判断；
- 是否有具体场景、图示或说明；
- 是否能支撑下一阶段继续推进；
- 是否避免只罗列工具名或空泛口号。
